# ChatQBJS

ChatQBJS is a conversational AI agent user interface for qbjs.org.

## Develop

Run the development server:

```sh
deno task start
```

Lint and format the code:

```sh
deno task precommit
```

Download few-shot examples:

```sh
deno task download
```
